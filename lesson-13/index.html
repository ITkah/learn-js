1. Press any key

Press any key on the keyboard.

The pressed key will appear here

2. Keyboard panel

Click the buttons on the panel and the symbols will appear on the screen.

3. Change background color with keys R, G, B

Press: R — red, G — green, B — blue.

Current color: default

4. Move the square with W, A, S, D keys

Use the W, A, S, D keys to move the square.

W — up, S — down, A — left, D — right
